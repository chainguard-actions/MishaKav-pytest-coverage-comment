# MishaKav/pytest-coverage-comment

GitHub Action that adds pytest coverage reports as comments to pull requests with badges, test statistics, and direct file links. Requires pull-requests: write permission.

Hardened by [Chainguard](https://www.chainguard.dev) from the upstream action at [https://github.com/MishaKav/pytest-coverage-comment](https://github.com/MishaKav/pytest-coverage-comment).
